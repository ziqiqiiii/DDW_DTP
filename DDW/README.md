## Getting started (Jupyter Notebook)
1. Create conda virtual environment
``` bash
conda create -n ddw_dtp python=3.12 anaconda mypy nb_mypy
```

2. Activate your ddw_dtp:
``` bash
conda activate ddw_dtp
```
